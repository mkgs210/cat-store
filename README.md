# cat store
### npm run dev - запуск сервера

### npm start - запуск клиента

Если по какой-то причине после запуска клиента бд не подключается, то можно запустить сервер в vscode или webstorm. Файл \server\index.js

## Описание
Сайт для аренды животных является площадкой, где клиент может найти различных животных для определенных нужд, например:
+ Приют бездомных животных
+ Аренда собак-поводырей маломобильными группами населения
+ Аренда животных для фотосессий
+ Аренда дрессированных животных

## Функционал
На сайте представлен следующий функционал:
+ Клиент:
  * Просмотр главной страницы
  * Выбор животных по критериям
  * Просмотр характеристик животных
  
+ Администратор:
  * Просмотр страницы администратора
  * Редактирование объявлений
  * Создание новых объявлений
  * Добавление новых городов
  * Добавление новых видов животных
